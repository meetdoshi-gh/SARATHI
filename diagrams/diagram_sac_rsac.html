Decision Layer — SAC vs RSAC Architecture Comparison
🎯 Soft Actor-Critic (SAC)
CNN Encoder (×3 instances)
Separate encoder per actor + per critic (×2). Gradient signals isolated — critic updates don't contaminate perception.
128-d per frame
Feature Fusion
Concatenate left (128) + right (128) + nav (10) → 266-d fused vector. One vector per timestep.
266-d
⟨ no recurrence ⟩
Single fused vector fed directly to actor / critic heads. Each timestep processed independently.
Actor Head — FC(512, 128)
266 → 512 → 128 → [µ, log_σ] (2-d each). Squashed Gaussian + reparameterization trick. Inference: tanh(µ).
2-dim continuous output
Critic Heads — FC(512, 128) × 2
[266-d state, 2-d action] → Q-value (scalar). Minimum of Q1, Q2 used in policy gradient (clipped double-Q).
scalar Q-value
Replay Buffer — Flat Transitions
Stores (s, a, r, s', done) tuples. Random uniform sampling. Buffer size = 50 epochs × 50 batches = 2500 batches.
batch_size=128
🔁 Recurrent SAC (RSAC) Novel
CNN Encoder (×3 instances)
Identical to SAC. Runs on each frame in the sequence. Separate encoder instances per actor/critic.
128-d per frame
Feature Fusion
Same concatenation. Produces 266-d vector at each timestep t in the sequence T. Shape: (T × 266).
T × 266-d sequence
LSTM Layer Added
Processes the T-step sequence of fused features. Hidden size h=128. Temporal context: T=16 steps. Hidden state initialized to zeros at sequence start.
h=128 · T=16
Actor Head — FC(h, h)
LSTM final hidden state (128-d) → [µ, log_σ]. Same squashed Gaussian policy. Same inference rule.
2-dim continuous output
Critic Heads — FC(h, h) × 2
LSTM final hidden + action sequence → scalar Q. Same clipped double-Q. Same Polyak target update.
scalar Q-value
Replay Buffer — Episode-Based Redesigned
Stores full episodes as lists of transitions. Samples contiguous fragments of length T=16. Ensures temporal coherence the LSTM relies on — not supported by standard RL libraries.
fragment sampling · T=16
Shared mechanics (both SAC and RSAC): Entropy auto-tuning — learnable log_α optimized against target entropy = −action_dim. Polyak averaging for target critics: θ_target ← τ·θ + (1−τ)·θ_target, τ=0.025. AdamW optimizer. γ=0.99. Actor and critics updated every training step.
