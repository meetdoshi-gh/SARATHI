RSAC
Extension
In Recurrent SAC (RSAC), an LSTM layer (h=128) is inserted between the fused feature vector and the actor/critic output heads. The standard transition replay buffer is replaced by an episode-based buffer that stores full rollouts and samples contiguous fragments of length T=16, preserving the temporal continuity the LSTM depends on. This architecture is built from scratch — not supported by standard RL libraries.
